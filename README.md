# GRC + SOC L1 Roadmap Tracker

A dark cybersecurity roadmap tracker for students preparing for GRC Intern, Junior GRC Analyst, SOC L1 Analyst, and Junior Information Security roles.

## Features

- Roadmap progress tracking
- XP and level system
- Current quest
- Certifications tracker
- Skills checklist
- Portfolio projects
- Job search tracker
- Interview prep
- LinkedIn builder
- Editor mode
- LocalStorage persistence
- JSON import/export
- Netlify-ready configuration

## Tech Stack

- React
- JavaScript
- CSS
- Vite
- LocalStorage

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

### Option 1: GitHub

1. Push the whole project to GitHub.
2. Connect the repository to Netlify.
3. Use these settings:

```txt
Build command: npm run build
Publish directory: dist
```

The included `netlify.toml` already sets this automatically.

### Option 2: Manual Upload

1. Run:

```bash
npm install
npm run build
```

2. Upload the generated `dist` folder to Netlify.
