# GRC + SOC L1 Roadmap Tracker

A dark cybersecurity roadmap tracker for students preparing for GRC Intern, Junior GRC Analyst, SOC L1 Analyst, Junior SOC Analyst, and Information Security Intern roles.

## Live Demo

https://arlanlab06.github.io/grc-soc-roadmap-tracker/

## Overview

GRC + SOC L1 Roadmap Tracker is a personal learning dashboard designed to make cybersecurity preparation more structured, visible, and portfolio-ready.

Instead of keeping roadmap tasks, certifications, skills, interview notes, job search steps, and portfolio ideas in different places, this app brings them together in one clean dashboard.

The project focuses on two beginner-friendly cybersecurity directions:

- GRC: Governance, Risk, and Compliance
- SOC L1: Security Operations Center Level 1

The goal is to help students and beginners track their learning progress, build practical portfolio evidence, and prepare for junior cybersecurity roles.

### Certifications

The certification tracker helps organize beginner-friendly certifications and courses, including Google Cybersecurity Certificate, Microsoft SC-900, Microsoft AZ-900, TryHackMe paths, and ISO-related learning.

### Skills

The skills checklist tracks GRC, SOC, and tool-based skills such as ISO 27001, NIST CSF, risk management, SIEM basics, phishing analysis, Microsoft security tools, and incident documentation.

### Interview Prep

The interview section includes common GRC and SOC L1 questions with structured answers for junior cybersecurity interviews.

### Job Search

The job search section helps track internship and junior role preparation tasks, target roles, and job platforms.

### LinkedIn Builder

The LinkedIn builder provides a cybersecurity-focused headline, about section, and skill list for improving a beginner cybersecurity profile.

### Editor

The editor allows roadmap sections, tasks, and portfolio items to be customized.
